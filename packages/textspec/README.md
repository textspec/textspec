# textspec

Convenience package that re-exports all [@textspec](https://www.npmjs.com/org/textspec) packages.

## Installation

```bash
npm install textspec
```

## Usage

```ts
import { parse, serialize } from "textspec/notation";
```

## Included packages

- [@textspec/notation](https://www.npmjs.com/package/@textspec/notation) - Parser and serializer for the Textspec editor state notation
